# Digital-Literacy
Submitted by: Vaibhav Ranjan (25BAS10060)

**Project Overview:**

i) A digital literacy project entails the development of skills that enable one to utilize digital technologies in an effective and responsible manner. 

ii) The project entails the ability to access and communicate information with the help of various digital technologies such as computers, smartphones, and the internet. The project covers various aspects such as online safety, cyber ethics, digital communication, and the ability to identify credible sources of information. 

iii) The project helps the students to learn the importance of protecting personal information and how to avoid cyber threats. In addition, the project helps the students to utilize digital technologies in a productive way for educational purposes. The project also helps the students to think critically in order to distinguish between credible and unreliable information. The project provides the students with an opportunity to learn and explore various digital technologies and tools, which helps them in the learning process.

5. Cybercrime- This module focused on the real world problems which prevail in the digital age. There are many issues which have become alarming some of them are Phishing, UPI Scams, OTP scams and many more. We learnt about various such cases and then focused on the solutions which can be implemented in our own lives so that we can be more aware and don't fall victim of such scams.
